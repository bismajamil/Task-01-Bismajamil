# 🤖 Rule-Based Chatbot

A simple and interactive **Rule-Based Chatbot** built using Python.  
This project uses **if-else logic and pattern matching** to understand user input and generate appropriate responses.

The chatbot can handle greetings, basic conversations, help commands, and exit commands.

---

## 📌 Project Overview

This project demonstrates the basic concept of a chatbot without using Artificial Intelligence models.

The chatbot works by:

1. Taking user input
2. Cleaning and normalizing the input
3. Matching input with predefined patterns
4. Returning a suitable response

---

## ✨ Features

✅ Greeting conversation  
- hello
- hi
- hey
- good morning

✅ Personal interaction
- Who are you?
- What is your name?
- How are you?

✅ Help system
- Shows available chatbot commands

✅ Exit commands
- bye
- exit
- quit

✅ Error handling

✅ Clean and modular Python code
